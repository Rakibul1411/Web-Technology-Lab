Structural/Sectioning Elements

Article

This represents independent content that could be distributed or syndicated.

Aside

This content is tangentially related to the content around it.

Details

Additional details that can be viewed or hidden.

Footer

This defines the footer for a document or section.

Header

This defines the header for a document or section.

Main

This represents the main content of a document.

Highlighted text

Navigation

Defines navigation links.

Section

Defines a thematic grouping of content.
