 Starting and Stopping Commands

 Most HTML commands come in pairs: for example, "<H4>" marks the
 beginning of a size 4 heading, and "</H4>" marks the end of it. The closing
 command is always the same as the opening command, except for the
 addition of the "/".

 Modifiers are sometimes included along with the basic command, inside the
 opening commands < >. The modifier does not need to be repeated in the
 closing command.
